Instructions to run locally:


1. Have node.js installed
2. In the command prompt, cd to the server folder, and type npm install 
3. In the command prompt, cd to the client folder, and type npm install 
4. After npm install for server has been completed, cd to the server folder, and type npm start
5. After npm install for client has been completed, cd to the client folder, and type npm start
6. Application will open at localhost:3000
